  • Solar Power Converter

    Currently in the process of designing a solar power converter. I will use a Boost DC Converter to step-up the voltage from the PV cell, and that voltage will be converted for usage in a utility grid with a DC to AC Boost Converter. I will also be implementing the controls for an MPPT in the design in order to maximize the efficiency of the solar converter.

  • Heart Sync App

    On-going senior project to develop an Android fitness application that interfaces with a bluetooth heart-rate monitor. The application will detect the user's heart-rate and play music accordingly. Our application will also take the user's daily resting heart rate in order to track their long-term workout progress.

  • Ping-pong on FPGA Board

    Implemented a ping-pong game using system level Verilog and an FPGA board. We designed the game so that the controls for the paddles would be manipulated by the push buttons on the FPGA board.


  • Tic-Tac-Toe

    Our embedded systems project was to implement a Tic Tac Toe game in C using the Code Composer Studio software for the LCD touchscreen boosterpack (TM4C Microcontroller). This project provided a great learning experience in both navigating the functions of the software and designing algorithms to create a computer game. Our code has different modes of play, including player versus player and player versus computer, with three different difficulty settings for the latter. Our code also implemented an interrupt in order to use the hibernation mode of the game.

  • Audio Headphone Amplifier PCB Board

    Created the PCB schematic for a low-noise audio headphone amplifier, including the system power supply, a two-stage amplifier circuit with discrete output stage, input overloading and output short protection, including panel mount user controls.
